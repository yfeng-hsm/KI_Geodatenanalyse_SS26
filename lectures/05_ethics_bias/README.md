# Lecture 05: Ethics and Bias

## Role in the Course

This lecture focuses on responsible AI for geospatial applications. It connects technical model choices to fairness, bias, accountability, and societal impact.

## Planned Topics

- Data bias in geospatial datasets
- Spatial sampling bias and representation gaps
- Fairness and proxy variables
- Privacy and sensitive location data
- Model accountability and documentation
- Uncertainty communication
- Responsible use of LLMs and automated spatial analysis

## Possible Geospatial Examples

- Platform bias in Airbnb data
- Coverage bias in Mapillary imagery
- Reporting and infrastructure bias in accident data
