# Lecture 03: LLM Basics

## Role in the Course

This lecture introduces large language models and connects them to geospatial analysis workflows.

## Planned Topics

- Tokenization and embeddings
- Pretraining and instruction tuning
- Prompting and structured outputs
- Retrieval-augmented generation
- Tool use and agents
- Limitations, hallucination, and evaluation
- Geospatial question answering and data interaction

## Possible Geospatial Examples

- Natural-language interfaces for spatial datasets
- Summarizing neighborhood-level patterns
- Combining LLMs with structured geospatial tools
