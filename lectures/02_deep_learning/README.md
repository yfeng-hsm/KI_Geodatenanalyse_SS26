# Lecture 02: Deep Learning

## Role in the Course

This lecture introduces neural networks and the main deep learning architectures relevant for geospatial AI.

## Planned Topics

- Neurons, layers, activations, and loss functions
- Backpropagation and optimization strategies
- Regularization and generalization
- CNNs for image-like geospatial data
- Transformers for sequence, text, and multimodal tasks
- GNNs for graph-structured spatial data

## Possible Geospatial Examples

- Street-level image analysis with Mapillary
- Spatial graph modeling with road networks or points of interest
- Comparing CNN, Transformer, and GNN assumptions
