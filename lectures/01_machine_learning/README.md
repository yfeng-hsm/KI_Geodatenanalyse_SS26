# Lecture 01: Machine Learning Basics

## Role in the Course

This lecture introduces the core machine learning workflow for geospatial data analysis. The emphasis is on models that are easy to inspect and explain before moving to more complex deep learning methods.

## Planned Topics

- Supervised learning workflow
- Feature engineering for geospatial data
- Train, validation, and test splits
- Baseline models
- Decision trees
- Random forests
- Gradient boosting models
- Model evaluation and interpretation
