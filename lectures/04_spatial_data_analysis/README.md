# Lecture 04: Spatial Data Analysis

## Role in the Course

This lecture reviews common spatial data analysis concepts and tools that support the machine learning and AI workflows used throughout the course.

## Planned Topics

- Vector and raster data models
- Coordinate reference systems
- Spatial joins and overlays
- Buffers, distances, and nearest-neighbor analysis
- Spatial aggregation
- Network and accessibility analysis
- Spatial autocorrelation and clustering
- Common Python and GIS tools

## Possible Geospatial Examples

- Airbnb neighborhood analysis
- OSM-derived accessibility indicators
- Unfallatlas accident pattern analysis
